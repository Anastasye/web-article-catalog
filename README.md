# Web-каталог статей с поиском

## Описание проекта
Веб-приложение для создания персональных коллекций научных статей с возможностью поиска по автору, теме и ключевым словам.

## Технологии
- **Backend**: Spring Boot 3.2.4
- **Frontend**: Thymeleaf, CSS
- **База данных**: H2 (in-memory)
- **Безопасность**: Spring Security
- **Сборка**: Maven
- **Тестирование**: JUnit 5, Mockito

## Функциональные возможности
- ✅ Регистрация и аутентификация пользователей
- ✅ Загрузка PDF-файлов (до 10 MB)
- ✅ Управление библиографическими данными
- ✅ Поиск по автору, теме и ключевым словам
- ✅ Публичные профили пользователей
- ✅ Пагинация списков
- ✅ Валидация данных
- ✅ Загрузка аватаров
